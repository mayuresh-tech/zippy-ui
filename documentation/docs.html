Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs or grid items.

Example of Image Avatar

Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add image tag add class according to size required (avatar-image-lg, avatar-image-md, avatar-image-sm, avatar-image-xs).

Example of Text Avatar

You can use initial letters of user in Avatar as well. You need to include class avatar-text along with avatar and size class. (e.g. class="avatar avatar-text avatar-text-lg")

Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped

Example of Responsive Image

You can add class responsive-img to make your image fit the width of container. It's height will get adjusted by keeping the aspect ratio same. If you want to change the aspect ratio, you will have to crop the image.

Example of Round Image

You can add class round-img to make your image round shaped.

Form Components

There are form fields listed below and form validation is also styled.

Example of Active form fields

Add form-input-label class to labels and form-input-field for input element, form-input-textarea for textarea element (mulitple lines input).
Add required class to labels if the field is required, it will show star on the field

Example of Disabled / Read only form fields

Add disabled attribute to your form element to make them disable. No need to add any extra class.
Add readonly attribute to your form element to make them read-only. No need to add any extra class.

Example of Read only plain text form field

Add form-input-field plain-text to your form element to make them read-only.
